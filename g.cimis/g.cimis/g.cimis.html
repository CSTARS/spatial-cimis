NAME

g.cimis - Performs steps for Spatial CIMIS calculations

KEYWORDS

CIMIS evapotranspiration

SYNOPSIS

g.cimis
g.cimis --help
g.cimis [-lnB] [cmd=string[,string,...]] sec=string [arg=string[,string,...]] [make=string] [--help] [--verbose] [--quiet] [--ui]

Flags:

-l
List commands
-n
dry-run (Pass -n to make command)
-B
force (Pass -B to make command)
--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

cmd=string[,string,...]
Command
sec=string [required]
Spatial CIMIS command section
Default: g.cimis
arg=string[,string,...]
Additional args to pass to make, (eg. files=foo)
make=string
Make command
Default: make

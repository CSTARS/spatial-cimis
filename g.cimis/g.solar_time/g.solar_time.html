NAME

g.solar_time - Display various solar time variables for a solar mapset

KEYWORDS

CIMIS evapotranspiration

SYNOPSIS

g.solar_time
g.solar_time --help
g.solar_time [cmd=string] [rast=name] [--help] [--verbose] [--quiet] [--ui]

Flags:

--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

cmd=string
Command: solpos | sretr | ssetr | daynight | prev | risedayset | day_parms | sretr_parms | ssetr_parms
rast=name
Name of raster map

g.solar_time is a simple set of scripts that give some information about the rasters within a solar mapset, that are named by their timestamp. It's basically a simple method to determine if mapsets are in the day or night, and when their previous mapset was.

NAME

g.cimis-dwr.py - Peforms workflows for DWR GOES15 services; zipcode summaries.

KEYWORDS

CIMIS evapotranspiration, zipcodes

SYNOPSIS

g.cimis-dwr.py
g.cimis-dwr.py --help
g.cimis-dwr.py [-lnB] [cmd=string[,string,...]] sec=string [make=string] [--help] [--verbose] [--quiet] [--ui]

Flags:

-l
List commands
-n
dry-run (Pass -n to make command)
-B
force (Pass -B to make command)
--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

cmd=string[,string,...]
Command
sec=string [required]
Spatial CIMIS command section, (Makefile to use)
Default: g.cimis-dwr
make=string
Make command. If you want to use a different directory, for debugging eg. use 'make -I <dir>' to add that in.
Default: make --no-builtin-rules

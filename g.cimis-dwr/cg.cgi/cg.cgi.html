NAME

cg.cgi.py - Get weather information for specific dates and times.

KEYWORDS

CIMIS,etxml,evapotranspiration

SYNOPSIS

cg.cgi.py
cg.cgi.py --help
cg.cgi.py [-x] [date=string[,string,...]] srid=string item=string[,string,...] [point=string[,string,...]] [zipcode=string[,string,...]] [x=string[,string,...]] [y=string[,string,...]] bbox=string height=string width=string [--help] [--verbose] [--quiet] [--ui]

Flags:

-x
Output XML
--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

date=string[,string,...]
Date(s) of interest
srid=string [required]
EPSG SRID
Default: 3310
item=string[,string,...] [required]
Items of interest from (ETo,Rs,K,Rnl,Tx,Tn,U2)
Default: ETo,Rs,K,Rnl,Tx,Tn,U2,Rso
point=string[,string,...]
Point(s) to access
zipcode=string[,string,...]
Zipcode(s) of interest
x=string[,string,...]
Box Pointer X
y=string[,string,...]
Box Pointer Y
bbox=string [required]
Bounding Box (e,s,w,n)
Default: -400000,-650000,600000,450000
height=string [required]
Box Height
Default: 550
width=string [required]
Box width
Default: 500

NAME

cg.zipcode.summary.py - CIMIS specific command to create zipcode_summaries

KEYWORDS

CIMIS,evapotranspiration

SYNOPSIS

cg.zipcode.summary.py
cg.zipcode.summary.py --help
cg.zipcode.summary.py [-nnn] [delimiter=string] rast=string[,string,...] [nodata=string] [--help] [--verbose] [--quiet] [--ui]

Flags:

-n
Do not print header information
-n
Do not print count of pixels per zipcode
-n
Do not mapset name to output
--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

delimiter=string
delimiter for table
Default: ,
rast=string[,string,...] [required]
Default Rasters to include
Default: Tn,Tx,U2,ea,Gc,G,K,Rnl,ETo
nodata=string
What to replace your no-data values with
Default: *
